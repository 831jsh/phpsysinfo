Installation and Usage

  • Just extract phpSysInfo in your web directory
  • Create your config file: cp phpsysinfo.ini.new phpsysinfo.ini
  • Open a web browser

You can also access the XML or JSON API:

  • XML: /phpsysinfo/xml.php?plugin=complete
  • JSON: /phpsysinfo/xml.php?plugin=complete&json
  • JSONP: /phpsysinfo/xml.php?plugin=complete&jsonp&callback=getData
